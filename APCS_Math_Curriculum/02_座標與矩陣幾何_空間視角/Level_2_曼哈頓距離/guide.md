# Level 2：曼哈頓距離（棋盤距離）

## 🎯 核心觀念
**「計程車司機的距離」**
在充滿房子的城市裡，我們不能像小鳥一樣飛斜線，必須沿著格子街道走直角。

## 💡 教學重點 (10%)
*   **關鍵字**：絕對值、路徑長度。
*   **公式**：`|x1 - x2| + |y1 - y2|`。

## 🎲 不插電活動 (70%)
### 活動：機器人送貨
*   **規則**：
    1.  在方格紙上畫出障礙物（房子）。
    2.  起點 (1,1)，終點 (5,4)。
    3.  請問最少要走幾格？（不能走斜線）。
    4.  讓孩子數格子，發現不管怎麼繞（只要不走回頭路），步數都是一樣的。

## 💻 程式實作建議 (20%)
### Python / Scratch
*   **題目**：計算步數。
*   **實作**：輸入起點和終點座標，程式算出距離。

## 🔗 APCS 連結
*   **題型**：地圖搜尋、最短路徑問題的前身。
