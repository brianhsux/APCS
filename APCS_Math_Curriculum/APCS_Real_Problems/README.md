# APCS 歷屆實作題詳解與視覺化 (Real Problems)

這個資料夾收錄了 **APCS (大學程式設計先修檢測)** 的歷屆「實作題」真題。
我們不只提供解答，還將這些題目改編成 **Pygame Zero 視覺化版本**，讓學生能「看見」解題過程。

## 🚀 如何使用

1.  **先讀題**：每個檔案開頭都有完整的題目描述與範例輸入。
2.  **看演示**：執行 `pgzrun apcs_XX_name.py`，觀察程式如何一步步處理資料。
3.  **看程式碼**：程式碼中包含標準的 APCS 解題函式 (Solution) 與視覺化函式 (Visualization) 的對照。

## 💡 給學生的建議

APCS 實作題通常分為四級：
- 第 1 題：基本運算與邏輯 (必拿分)
- 第 2 題：字串或矩陣處理 (細心就能對)
- 第 3 題：演算法或遞迴 (需要思考)
- 第 4 題：複雜資料結構或 DP (挑戰題)

建議從第 1、2 題開始練習，建立信心後再挑戰第 3 題。
