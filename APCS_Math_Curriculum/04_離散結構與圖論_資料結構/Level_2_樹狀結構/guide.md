# Level 2：樹狀結構 (Tree)

## 🎯 核心觀念
**「族譜與資料夾」**
樹狀結構就是「有階級」的關係。這跟真實的樹是倒過來的，根 (Root) 在最上面。

## 💡 教學重點 (10%)
*   **關鍵字**：根 (Root)、葉 (Leaf)、父節點、子節點。
*   **觀念**：一個小孩只能有一個爸爸（在這種樹裡）。

## 🎲 不插電活動 (70%)
### 活動：家譜樹 / 檔案總管
*   **規則**：
    1.  畫出爺爺奶奶 -> 爸爸媽媽 -> 自己的關係圖。
    2.  或者用電腦打開檔案總管，一層一層點開資料夾，這就是 Tree 的走訪。

## 💻 程式實作建議 (20%)
### Scratch / Python
*   **題目**：猜數字遊戲（二分搜尋法概念）。
*   **實作**：1-100 猜數字，每次猜中間，這就是一棵二元搜尋樹 (BST) 的路徑。

## 🔗 APCS 連結
*   **題型**：樹的走訪 (Traversal)、遞迴操作。
