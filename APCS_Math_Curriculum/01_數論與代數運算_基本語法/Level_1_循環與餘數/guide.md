# Level 1：循環與餘數（模運算基礎）

## 🎯 核心觀念
**「世界是轉圈圈的」**
電腦不知道什麼是「星期幾」，但它知道數字數到 7 就要回到 1。這就是「餘數」的概念。在程式中，我們用 `%` (Mod) 來表示。

## 💡 教學重點 (10%)
*   **關鍵字**：週期、規律、分組、餘數。
*   **口訣**：「除不盡的剩下多少？」、「轉盤轉幾圈後停在哪？」

## 🎲 不插電活動 (70%)
### 活動一：紅綠燈木頭人
*   **規則**：
    1.  喊 1 (紅燈)、2 (黃燈)、3 (綠燈)、4 (紅燈)...
    2.  讓孩子發現：`數字 ÷ 3` 的餘數決定了燈號。
    3.  餘數 1 = 紅，餘數 2 = 黃，餘數 0 = 綠。

### 活動二：分組大風吹
*   **規則**：全班 20 人，每 3 人一組，最後剩下幾個人沒組別？這就是 `20 % 3 = 2`。

## 💻 程式實作建議 (20%)
### Scratch / Python
*   **題目**：製作一個會變色的紅綠燈。
*   **邏輯**：
    ```python
    # Python 概念
    time = 0
    while True:
        time = time + 1
        mode = time % 3
        if mode == 1:
            print("紅燈")
        elif mode == 2:
            print("黃燈")
        else:
            print("綠燈")
    ```
