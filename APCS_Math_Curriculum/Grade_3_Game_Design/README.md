# 小三 (Grade 3) 遊戲 Demo 一覽 (10 堂課)

依**教學順序**從易到難，對應座標、邏輯、迴圈等觀念。全部使用 **Pygame Zero**。

**執行方式：** `pgzrun 檔名.py`

| 順序 | 檔案 | 對應觀念 | 玩法 |
|------|------|----------|------|
| 1 | `demo_01_basic_move.py` | 座標 (x, y) | 基礎移動，認識螢幕座標系 |
| 2 | `demo_02_boundaries.py` | 邏輯 (if) | 加上邊界檢查，不能跑出畫面 |
| 3 | `demo_03_distance.py` | 距離 (abs) | 計算與寶箱的曼哈頓距離 |
| 4 | `demo_04_chase.py` | 比較 (Compare) | 鬼會自動追蹤玩家 (AI 雛形) |
| 5 | `demo_05_traffic_light.py` | 餘數 (Modulo) | 紅綠燈循環控制 |
| 6 | `demo_06_animation.py` | 動畫 (Frame) | 利用餘數製作跑步動畫 |
| 7 | `demo_07_shooting.py` | 向量 (Vector) | 發射子彈 (持續改變座標) |
| 8 | `demo_08_falling_rocks.py` | 亂數 (Random) | 躲避隨機掉落的石頭 |
| 9 | `demo_09_coins.py` | 列表 (List) | 收集畫面上的多個金幣 |
| 10 | `demo_10_maze_final.py` | 綜合應用 | 走迷宮 + 體力限制 (期末專題) |

---
*註：原 `demo_movement.py` 因需外部圖片已移除，請使用 `demo_01_basic_move.py` 代替。*
