# 小三 (Grade 3) 遊戲設計與數學觀念對照表

這個階段的核心是**「直覺與空間」**。
目標：讓孩子習慣用 (x, y) 思考，並理解電腦怎麼處理「重複」的事情。

## 🗺️ 模組一：地圖探索者 (Coordinates)
*   **對應數學**：`02_座標與矩陣幾何` - Level 1 & 2
*   **遊戲專題**：**《勇者走迷宮》**
*   **數學觀念應用**：
    1.  **定位**：螢幕就是一張大方格紙，勇者在 (100, 200)。
    2.  **移動**：按下右鍵 -> `x = x + 5`。
    3.  **距離**：計算離寶箱還有多遠 (曼哈頓距離 `abs(x1-x2) + abs(y1-y2)`)。

## 🔄 模組二：循環動畫師 (Modulo)
*   **對應數學**：`01_數論與代數運算` - Level 1
*   **遊戲專題**：**《紅綠燈與跑步動畫》**
*   **數學觀念應用**：
    1.  **循環**：利用餘數 `%` 讓圖片輪播 (0, 1, 2, 0, 1, 2...)。
    2.  **計時**：每 60 秒 (Frames) 換一次背景。
