# APCS 數學思維教學地圖 (For 3rd-4th Grade)

這是一個專為三、四年級學生設計的 APCS 數學先修課程結構。
我們的目標不是背誦公式，而是建立**數感**與**空間感**。

## 📂 資料夾結構說明

本課程分為四大模組，每個模組由淺入深分為 4 個 Level：

1.  **01_數論與代數運算**：程式的基本語法（循環、二進位、質數、GCD）。
2.  **02_座標與矩陣幾何**：程式的空間視角（座標、距離、矩陣、向量）。
3.  **03_組合計數與邏輯**：程式的效率核心（排列組合、規律、遞迴）。
4.  **04_離散結構與圖論**：程式的資料結構（集合、樹、圖、最短路徑）。

## 📝 教學黃金比例

*   **70% 不插電活動**：透過遊戲、卡牌、身體律動來理解邏輯。
*   **20% 程式實作**：使用 Scratch 或簡單 Python 驗證想法。
*   **10% 數學觀念**：最後才帶入數學名詞與公式。

## 🚀 如何使用
請進入各個 Level 資料夾，閱讀 `guide.md` 獲取詳細的教學指引與題目設計。
