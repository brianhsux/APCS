# 小四 (Grade 4) 遊戲 Demo 一覽 (10 堂課)

依**教學順序**從易到難，對應邏輯、陣列、演算法等觀念。全部使用 **Pygame Zero**。

**執行方式：** `pgzrun 檔名.py`

| 順序 | 檔案 | 對應觀念 | 玩法 |
|------|------|----------|------|
| 1 | `demo_01_logic_gates.py` | 邏輯 (AND/OR) | 可視化開關電路，理解「且」的運算 |
| 2 | `demo_02_inventory_logic.py` | 集合 (Set) | 撿道具與檢查背包內容 (in / not in) |
| 3 | `demo_03_grid_walker.py` | 陣列 (Array) | 基礎二維地圖，理解 0 是路 1 是牆 |
| 4 | `demo_04_map_editor.py` | 修改陣列 | 點擊格子改變地圖資料 (0 變 1) |
| 5 | `demo_05_probability.py` | 機率 (Random) | 抽獎機率模擬 (區間判斷) |
| 6 | `demo_06_matrix_battle.py` | 索引 (Index) | 數字鍵 1-9 對應 (row, col) 座標 |
| 7 | `demo_07_binary_guess.py` | 二分搜尋法 | 電腦猜數字，視覺化 High/Low 範圍 |
| 8 | `demo_08_flood_fill.py` | 遞迴 (Recursion) | 油漆桶填色工具 (擴散演算法) |
| 9 | `demo_09_bubble_sort.py` | 排序 (Sort) | 視覺化泡泡排序過程 (兩兩交換) |
| 10 | `demo_10_rpg_battle.py` | 狀態機 (State) | 回合制戰鬥系統 (你一拳我一拳) |
